# Troubleshooting

## The plugin opens but doesn’t create a flow
- Ensure **Experiment name** is filled
- Try **Refresh** if a flow already exists

## Fonts look wrong
Install the **Figtree** font and restart Figma.

## Winner badge not showing
- Set status to **Concluded**
- Select a rolled-out variant
- Mark the winner in the variants list

## Can’t add variants
Only the **Experiment Step** touchpoint can contain variants. Mark the step first.

## Links show generic icon
Use full URLs (include https://). Unsupported services fall back to a generic icon.
