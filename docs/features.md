# Features

## Experiment documentation
- Name, description, hypothesis, status
- Timeline, audience, sample size, owner
- Advanced settings for metadata

## Goals & metrics
- Multiple metrics with primary designation
- Direction (increase/decrease) and thresholds
- Drag-and-drop ordering

## Journey mapping
- Entry and exit nodes
- Touchpoints with experiment step designation
- Connected visual flow spine

## Variants
- Unlimited variants and traffic splits
- Control and winner badges
- Color badges and ordering
- Create from selection (up to 3 frames)

## Resource links
- Smart link detection with service icons
- Figma, Jira, Miro, Notion, Asana, Linear, GitHub, Slack, Trello, Confluence, Monday, ClickUp
