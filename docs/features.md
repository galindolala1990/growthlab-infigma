# Features

Growthlab Builder is packed with features to help you document experiments efficiently and professionally.

## Core Features

### 🎯 Experiment Information

Capture all the essential details:
- **Experiment name** (required)
- **Description** - What you're testing
- **Status** - Running, Concluded, Planned, Draft, Paused
- **Hypothesis** - Your if-then-because statement
- **Timeline** - Start and end dates
- **Audience** - Target user segment
- **Sample size** - Number of users
- **Owner** - Responsible person or team
- **Confidence level** - Statistical confidence (95% default)

### 📊 Goals & Metrics

Define what success looks like:
- Add multiple metrics
- Mark one as **primary** (starred)
- Drag and drop to reorder
- Visual hierarchy in generated diagrams

**Common metrics:**
- Conversion rate
- Click-through rate (CTR)
- Revenue per user
- Time on page
- Bounce rate

### 🗺️ Journey Mapping

Visualize the complete user flow:
- **Entry node** - Where users start
- **Touchpoints** - Steps in the journey
- **Experiment step** - Where testing happens
- **Exit node** - Where users complete or exit

**Experiment Step:**
- Mark one touchpoint as the experiment location
- Add variants only to this step
- Keeps diagrams focused and clear

### 🎨 Variant Management

Full control over test variations:
- **Control variant** - Always included (baseline)
- **Add variants** - A, B, C, D... as many as needed
- **Traffic splits** - Set percentage for each
- **Winner badges** - Mark winning variants (for concluded experiments)
- **Visual thumbnails** - Create from selected Figma frames

**Supported test types:**
- A/B tests (2 variants)
- A/B/C tests (3+ variants)
- Multivariate tests (multiple variants)

### 🔄 Update Existing Flows

Made changes? No problem:
1. Make edits in the plugin
2. Click **"Refresh"** instead of "Create flow"
3. Existing diagram updates automatically

No need to delete and recreate!

### 🎨 Professional Styling

Built on Growth Labs design system:
- **Figtree font** - Clean, modern typography
- **Design tokens** - Consistent colors and spacing
- **Auto Layout** - Easy to resize and customize
- **Component-based** - Modular structure

### 🚀 Performance

Fast and efficient:
- Generates flows in seconds
- Works with large files
- Minimal impact on Figma performance

## Diagram Components

Your generated flow includes:

### 1. Experiment Info Card
Top card with all metadata:
- Experiment name and description
- Status badge with color coding
- Timeline (start/end dates)
- Owner and sample size
- Resource links with icons
- All metrics/goals

### 2. Journey Flow
Visual user flow:
- Entry node (circle)
- Touchpoint cards
- Experiment step with variants
- Traffic split indicators
- Exit node (circle)
- Connectors between all elements

### 3. Outcome Cards
Results per variant:
- Variant name
- Traffic percentage
- Winner badge (if applicable)
- Control indicator
- Aligned with journey flow

## Workflow Features

### Quick Start Mode
Minimal input to get started:
- Just add experiment name
- Get default 2-variant A/B test
- Customize later

### Advanced Mode
Complete experiment documentation:
- All metadata fields
- Multiple metrics and variants
- Resource links
- Detailed hypothesis and audience

### Create from Selection
Visual variant creation:
1. Design variants as Figma frames
2. Select up to 3 frames
3. Click "Create from selection"
4. Auto-generates variants with thumbnails

Perfect for visual experiments!

### Status Tracking
Track experiment lifecycle:
- **Draft** - Work in progress
- **Planned** - Scheduled but not started
- **Running** - Currently live
- **Paused** - Temporarily stopped
- **Concluded** - Finished (can mark winner)

## Integration Features

### Figma Integration
Native Figma features:

Built for teams:
