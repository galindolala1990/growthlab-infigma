# Plugin UX Guidelines

## Official Docs
- Plugin UI best practices: https://www.figma.com/plugin-docs/intro/#designing-the-ui

## What to focus on
- Clear labels and simple flows
- Small, fast interactions
- Helpful empty states
- Consistent spacing and typography
